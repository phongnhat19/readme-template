### How to build
```
$ git clone https://github.com/kintone/kintone-js-sdk.git
$ cd kintone-js-sdk
$ npm install
$ npm run build
```

Output

```
/dist/kintone-js-sdk.min.js
/esm/browser/main.js
/cjs/node/main.js
```
